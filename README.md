# Amana Health - Modern Healthcare Platform

This is a Next.js application, demonstrating a modern healthcare platform.

## Key Features

*   **Patient Dashboard**: A comprehensive dashboard for patients to manage their health information, appointments, and messages.
*   **Provider Dashboard**: A dedicated interface for healthcare providers to manage patients, schedules, and availability.
*   **Appointment Booking**: A multi-step, user-friendly appointment booking system.
*   **AI-Powered Diagnosis**: An intelligent diagnosis tool to get suggestions based on symptoms.
*   **Secure Messaging**: A messaging system for patients and doctors to communicate securely.
*   **Provider & Patient Profiles**: Detailed profiles for both doctors and patients.
*   **Search Functionality**: Search for doctors and facilities with advanced filters.
*   **Billing & Subscription Management**: Pages for users to manage their payments and subscriptions.
*   **Pharmacy Shop**: An e-commerce-like page for browsing medical products.

## Tech Stack

*   **Framework**: [Next.js](https://nextjs.org/) (App Router)
*   **UI**: [React](https://react.dev/), [TypeScript](https://www.typescriptlang.org/)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/), [shadcn/ui](https://ui.shadcn.com/)
*   **Generative AI**: [Genkit](https://firebase.google.com/docs/genkit) for AI flows.
*   **Authentication & Database**: (Ready for Firebase integration)

## Getting Started

To get started with the application, explore the main patient-facing page at `src/app/page.tsx` or the patient dashboard at `src/app/dashboard/page.tsx`.

The provider-side of the application starts at `src/app/provider/page.tsx`.
